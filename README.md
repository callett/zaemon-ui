# ✨ 项目简介

这是一个基于 Fuwari 改造的个人博客项目，采用极简主义设计风格，支持响应式布局，适合用于记录技术文章、生活随笔或开发笔记等内容。

本项目在原始 Fuwari 模板的基础上进行了定制优化，增强了使用体验和样式细节，更加符合个人写作习惯。

⸻

# 🔧 特性一览
	•	🌸 继承 Fuwari 的简洁优雅风格
	•	📱 响应式设计，支持 PC 和移动端访问
	•	📄 支持 Markdown 编写文章
	•	🏷️ 支持标签分类（如有此功能）
	•	🌙 可选深色模式（如已添加）
	•	🚀 快速部署，支持静态托管（GitHub Pages / Cloudflare Pages 等）
